# 🧮 Timed Math Challenge

A fun and interactive command-line math game built using Python. Test your arithmetic skills by solving as many math problems as possible within the time limit.

## 🚀 Features

- ⏱️ 30-second timer
- ➕ Addition questions
- ➖ Subtraction questions
- ✖️ Multiplication questions
- ➗ Division questions (integer answers only)
- 🎯 Score tracking
- ✅ Correct answer counter
- ❌ Wrong answer counter
- 🔄 Play Again option
- 📄 Single Python file
- 🐍 Beginner-friendly project

## ▶️ How to Run

### Clone the repository

```bash
git clone https://github.com/your-username/Timed-Math-Challenge.git
```

### Go to the project folder

```bash
cd Timed-Math-Challenge
```

### Run the program

```bash
python timed_math_challenge.py
```

---

## 🎮 Gameplay

- You have **30 seconds** to answer as many questions as possible.
- Questions include:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Each correct answer earns **10 points**.
- The game ends automatically when the timer reaches zero.
